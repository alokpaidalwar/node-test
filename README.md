# Nodejs api - stopwatch logs

### Clone the repo

```javascript 
git clone https://github.com/alokpaidalwar/node-test.git
cd node-test
npm install
```

### Add env details

Add dburl and port in .env file

### Start the application
 
 To run in dev mode

 ``` 
 npm run dev
 ```

 To run in prod mode

 ```
 npm run start
 ```
